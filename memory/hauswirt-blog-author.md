# Hauswirt Blog 作者

所有 Hauswirt Blog 文章统一使用作者：**Ethan Brooks**

- 发布脚本 `scripts/publish_blog.py` 默认作者 = "Ethan Brooks"
- 文章 byline 必须与此一致，不得使用 "Hauswirt Editorial" 或其他变体
- 目的：EEAT 中的 Authoritativeness 信号 — 统一 byline 建立作者权威
