## 战略调整

原 Phase 2/3（6月4篇 + 7-9月6篇）→ 新规划：5月追加5篇 / 6-11月 50篇，合计 55 篇新文章，目标关键词覆盖率 92.5%（750+/811）。

新增内容方向（调研驱动）：
- 猫砂与哮喘关系（Reddit 高频，零竞品内容）
- 急救指南（Google "when to worry" 高搜索量）
- 费用透明化（Reddit 用户核心焦虑）
- 自然疗法循证评估（Reddit 争议话题，高互动）
- 情绪支持 + 兽医沟通清单（新确诊猫主需求）
- **Amazon 评论驱动（2026-05-17 新增）**：呼吸指示阀故障排查、间隔器干燥指南、价格对比——基于 AeroKat 100条 + Neobay 19条评论分析，直接拦截"买便宜货→翻片不动→换回AeroKat"流失路径
- 每篇写前必做 SERP TOP 10 竞品结构分析

## 执行注意事项

1. **每篇写前必须 Phase 0 调研**（Google Autocomplete + Reddit + SERP TOP 10），已写入 `neobay-blog` SKILL.md
2. **产品卡插入**：转化层必含 `%%bpc:spacer%%`，知识层视情况，症状层不强制
3. **内链**：参照 [[neobay-content-strategy]] 5 大 Cluster + 6 条规则
4. **Blog 分类**：哮喘医学→Feline Asthma、呼吸症状→Cat Breathing、设备→Spacer Guides
5. **月度复盘**：每月最后一周检查排名/流量/索引，必要时调整下月选题
6. **68篇为总上限**：5月18篇+6-11月50篇，不包含临时热点或产品公告，需插入可替换同 Cluster 低优先级选题
7. **Amazon 评论分析（2026-05-17）**：已导出并分析 AeroKat 100条 + Neobay 19条评论。核心发现：呼吸指示阀是竞争主战场（AeroKat #1卖点 = Neobay #1差评源）；"先买便宜→翻片不动→换AeroKat"是已验证流失路径；干燥困难是双方共同痛点；AeroKat 价格/配件政策是主要进攻空间。据此替换了 #51/#58/#61 为 3 篇高转化新选题，调整了 #21/#30/#36 内容重点
8. 完整选题表见 `Neobay/docs/research-report-blog-plan-2026.md`
