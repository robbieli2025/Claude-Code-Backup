# 用户费用类型白名单

每个用户定义自己需要报销的费用类型。只录入白名单内的服务商。

## 如何编辑

直接在下方按格式添加/修改用户配置。每个用户一个 `###` 段落。

服务商用 **飞书 Base 中实际使用的服务商名称**，与截图提取后校验。
