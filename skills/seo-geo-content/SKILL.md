# SEO GEO Content

## Purpose

Use this skill for content that must perform in Google search and be easy for AI search systems to parse, cite, and summarize.

## Workflow

1. Start with the business and page goal.
   - Identify target audience, offer, product category, market, language, and conversion goal.
   - Separate informational, commercial, transactional, and navigational intent.
   - Ask for brand voice, competitors, or source material only when the task cannot be completed responsibly without them.

2. Build the search strategy.
   - Map one primary query intent per page.
   - Add secondary keywords, related questions, comparison angles, objections, and internal link targets.
   - Avoid keyword stuffing; prioritize clear answers, useful detail, and scannable structure.
   - For current SEO rules or claims about Google policy, verify against official Google Search Central or current primary sources when possible.

3. Structure for Google SEO and AI GEO.
   - Use a clear H1 that matches the page promise.
   - Use H2/H3 sections that answer concrete subquestions.
   - Put concise answer passages near the relevant heading.
   - Include entity-rich descriptions, product/category context, specs, benefits, use cases, limitations, and comparison criteria where relevant.
   - Add FAQ only when questions are genuinely useful.
   - Suggest schema types only when they accurately match the page content.

4. Write metadata and page copy.
   - Title tags should usually be under about 60 characters when possible, front-load the main topic, and avoid empty clickbait.
   - Meta descriptions should usually be under about 155 characters when possible, summarize the page clearly, and include a reason to click.
   - Blog and landing copy should be original, specific, and aligned with search intent.
   - For ecommerce pages, connect copy to product attributes, decision criteria, trust signals, shipping/returns concerns, and category navigation.

5. Quality check.
   - Check duplication across pages, unclear claims, thin sections, missing internal links, weak headings, and unsupported superlatives.
   - Flag where expert quotes, product data, customer proof, images, or schema implementation would improve credibility.
   - Deliver content in a format the user can paste into Shopify, a CMS, a spreadsheet, or a brief.

## Output Expectations

For SEO/GEO work, prefer structured outputs such as:
- Page goal and search intent.
- Recommended URL slug.
- SEO title and meta description.
- H1-H3 outline.
- Draft copy or blog post.
- FAQ and schema suggestions.
- Internal link recommendations.
- Notes on evidence, images, and trust signals needed.
